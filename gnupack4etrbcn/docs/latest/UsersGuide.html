最初にお読みください．

　gnupackはアーカイブ展開のみで即座に cygwinと emacsが使える環境を提供します．gnupackの配布ライセンスは GPLv2です．

　開発者の動作確認は WindowsXP 32bit環境のみになっています．Windows95/98/98SE/Meは動作しない可能性があります．Windows Vista以降の OSバージョンについては動作する可能性はありますが，サポート外になります．gnupackのバグ改修や機能追加等のバージョンアップは本パッケージの開発者の対応可能な範囲内，かつ開発者の作業ペースにて行わせていただきます．

　最新の gnupack Users Guideは http://gnupack.sourceforge.jp/docs/latest/UsersGuide.htmlを参照ください．なお，本ページはFirefox 6.0（画面幅：950px，フォント：MeiryoKe_PGothic/MeiryoKe_PGothic 12pt）を基準にデザインしています．その他のブラウザで正しく表示できない可能性があります．

gnupackの改版内容

■バグ改修・機能改善
  - cygwinで環境変数 INST_DIRが認識できない問題を暫定対処（INST_DIR=$inst_dirを追加）．
  - emacsの find-file実行時の補完がおかしくなる問題を対処（file-name-shadow-mode nilに設定）．
  - elisp package menu-treeで emacs24において日本語化されない項目を修正．

■設定変更
  - elisp package tabbarの配色を Windowsのテーマに合わせるように変更．

■パッケージ構成変更
