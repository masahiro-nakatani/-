========================================
               NxtLogger
      Nxtログ取得アプリケーション
========================================


■ 動作条件
　　.Net Framework 3.5 以上

■ 動作検証環境
　　Windows 7 Professional

■ 開発環境
　　Visual Studio C# 2008

■ 動作手順
 1. PCとNXTのSPP(Bluetooth)接続を確立する。
 2. NxtLoggerでSPP接続したポートを選択する。
 3. NxtLoggerでCONNECTをクリックする。
 4. CTSとDSRがアクティブになるのを待つ。
 5. NXTからのログ送信を開始する。

 ※ ログファイルは自動的に保存されます。
